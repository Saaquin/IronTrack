# IronTrack
High-performance headless Rust engine for automated aerial survey flight planning. Precision Karney geodesics, 6th-order Karney-Krüger UTM, and Copernicus GLO-30/EGM2008 terrain awareness. Exports terrain-adjusted 3D flight lines to GeoPackage and GeoJSON. Built for memory safety, high concurrency, and survey-grade accuracy.
